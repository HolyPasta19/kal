# Crosshair Overlay с GUI

Приложение с визуальным интерфейсом для настройки прицела.

## Установка

```cmd
pip install -r requirements.txt
```

## Запуск

```cmd
python crosshair_gui.py
```

## Возможности

### Вкладка "Прицел"
- Настройка цвета через RGB слайдеры (0-255)
- Настройка прозрачности (0-255)
- Настройка размера, толщины линий и отступа
- Выбор типа прицела:
  - Точка
  - Крест
  - Т-образный
  - Капля
  - Квадрат
  - Окружность

### Вкладка "Кастомное изображение"
- Чекбокс для включения кастомного изображения
- Browse для выбора файла изображения
- Предпросмотр выбранного изображения
- Настройка размера изображения
- Настройка прозрачности
- RGB маска для удаления определенного цвета с изображения

### Вкладка "Хоткеи"
- Настройка клавиши для включения/выключения прицела
- Кнопка "Записать" для удобного выбора клавиши

## Использование

1. Запустите приложение - прицел появится в центре экрана
2. Настройте прицел в GUI окне
3. Все изменения применяются мгновенно
4. Нажмите хоткей (по умолчанию F1) для включения/выключения прицела
5. Настройки сохраняются автоматически в config.json

## Примечания

- Прицел всегда отображается поверх всех окон
- При включении кастомного изображения стандартный прицел отключается
- RGB маска позволяет убрать фон с изображения (например, розовый #FF00FF)
- Прозрачность работает для обоих режимов
