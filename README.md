# Crosshair Overlay + Magnifier

Overlay-прицел и экранная лупа для игр с GUI настройками.

## Возможности

### Прицел
- 6 типов прицелов: точка, круг, круг с точкой, крест, шеврон, крест без точки
- Настройка цвета (RGB), прозрачности, размера, толщины, отступа
- Click-through (клики проходят сквозь окно)
- Всегда поверх всех окон
- Горячие клавиши для включения/выключения

### Лупа
- Захват области экрана вокруг центра с увеличением
- Настройка размера окна, кратности приближения (1x-8x), FPS (15-120)
- Click-through и исключение из захвата экрана
- Отдельный процесс для стабильной работы
- Горячие клавиши для включения/выключения

### Горячие клавиши
- Поддержка клавиатуры: буквы, цифры, F1-F12, спецсимволы (-, =, \)
- Поддержка мыши: mouse1-5, scroll_up, scroll_down
- Модификаторы: Ctrl, Alt
- Работают независимо от зажатых Shift/Ctrl/Alt

## Установка

```bash
pip install -r requirements.txt
```

## Запуск

```bash
python crosshair_gui.py
```

## Сборка в .exe

```bash
python -m PyInstaller --onefile --windowed --icon=Sprite-0001.ico --add-data "Sprite-0001.ico;." --name=Crosshair --clean crosshair_gui.py
```

Результат: `dist/Crosshair.exe`

## Использование

1. Запустите приложение
2. Настройте прицел во вкладке "Прицел"
3. Настройте лупу во вкладке "Лупа"
4. Назначьте горячие клавиши во вкладке "Хоткеи"
5. Используйте горячие клавиши для включения/выключения

## Конфигурация

Настройки сохраняются в `config.json`:

```json
{
  "hotkeys": {
    "toggle": "f1",
    "magnifier": "mouse4"
  },
  "crosshair": {
    "type": "cross",
    "r": 0,
    "g": 255,
    "b": 0,
    "alpha": 255,
    "size": 15,
    "thickness": 2,
    "gap": 5,
    "dot_size": 2
  },
  "magnifier": {
    "display_size": 300,
    "capture_size": 150,
    "zoom": 2.0,
    "target_fps": 60
  }
}
```

## Зависимости

- tkinter (GUI)
- PyQt5 (лупа)
- Pillow (изображения)
- pynput (хоткеи)
- mss (захват экрана)
- opencv-python (масштабирование)
- numpy (обработка изображений)
