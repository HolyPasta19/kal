# Kal - Crosshair Overlay & Magnifier

Настраиваемый прицел и лупа для игр с темной темой интерфейса.

## Возможности

### Прицел
- **6 типов прицелов**: точка, круг, круг с точкой, крест, шеврон, крест без точки
- **Полная настройка**: цвет (RGB), прозрачность, размер, толщина, отступ
- **Алгоритм Брезенхэма**: идеально ровные круги без искажений
- **Прозрачное окно**: не мешает игровому процессу
- **Горячие клавиши**: быстрое включение/выключение

### Лупа
- **Увеличение центра экрана**: от 1x до 8x
- **Настраиваемый размер окна**: 100-1000 пикселей
- **Высокая производительность**: 15-120 FPS
- **Режимы интерполяции**: nearest, linear, cubic, lanczos
- **Исключение из захвата**: лупа не видна на скриншотах и в записи
- **IPC управление**: работает как отдельный процесс

### Интерфейс
- **Темная тема**: современный темный интерфейс
- **Превью в реальном времени**: видишь изменения сразу
- **Поддержка мыши**: хоткеи на кнопки мыши (mouse1-5) и колесико
- **Модификаторы**: Ctrl/Alt для комбинаций

## Установка

### Из исходников

1. Установи зависимости:
```bash
pip install -r requirements.txt
```

2. Запусти:
```bash
python crosshair_gui.py
```

### Сборка установщика

1. Собери exe:
```bash
python -m PyInstaller --onedir --windowed --name="Kal" --icon="Sprite-0001.ico" --add-data "Sprite-0001.ico;." crosshair_gui.py
```

2. Скачай и установи [Inno Setup](https://jrsoftware.org/isdl.php)

3. Открой `installer.iss` в Inno Setup Compiler

4. Нажми F9 для компиляции

5. Установщик будет в папке `installer/Kal_Setup.exe`

## Использование

1. Запусти программу
2. Настрой прицел во вкладке "Прицел"
3. Настрой лупу во вкладке "Лупа"
4. Назначь хоткеи во вкладке "Хоткеи"
5. Используй хоткеи в игре для включения/выключения

## Горячие клавиши

- Поддерживаются все клавиши клавиатуры (F1-F12, буквы, цифры, спецсимволы)
- Кнопки мыши: mouse1-5
- Колесико мыши: scroll_up, scroll_down
- Модификаторы: Ctrl, Alt (можно комбинировать)

## Безопасность

⚠️ **Важно**: Использование overlay программ может быть обнаружено античитами.

**Безопасно для**:
- Казуальные игры
- Одиночные игры
- Игры без античита

**НЕ рекомендуется для**:
- Valorant (Vanguard)
- FACEIT / ESEA
- Турниры и соревнования
- Ranked режимы с агрессивными античитами

Программа не модифицирует игровые файлы и не читает память процессов, но использует API для захвата экрана и overlay, что может быть расценено как нарушение.

## Технические детали

- **Язык**: Python 3.14
- **GUI**: tkinter + PyQt5
- **Захват экрана**: mss
- **Обработка изображений**: OpenCV, NumPy
- **Ввод**: pynput

## Конфигурация

Настройки сохраняются в `config.json`:

```json
{
  "hotkeys": {
    "toggle": "ctrl+1",
    "magnifier": "\\"
  },
  "crosshair": {
    "type": "chevron",
    "r": 0, "g": 255, "b": 253,
    "alpha": 255,
    "size": 14.75,
    "thickness": 4.75,
    "gap": 12.06,
    "dot_size": 2.1
  },
  "magnifier": {
    "display_size": 1000,
    "zoom": 3.0,
    "target_fps": 61,
    "interpolation": "nearest"
  }
}
```
