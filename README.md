# Crosshair Overlay

Приложение для отображения прицела и экранной лупы поверх всех окон в играх.

## Установка

```cmd
pip install -r requirements.txt
```

## Запуск

**С GUI (рекомендуется):**
```cmd
python crosshair_gui.py
```

**Только прицел (без GUI):**
```cmd
python crosshair.py
```

**Только лупа (для тестирования):**
```cmd
python magnifier.py
```

## Настройка (config.json)

### Горячие клавиши
- `toggle` - включить/выключить прицел (по умолчанию F1)
- `exit` - закрыть приложение (по умолчанию Escape)

### Параметры прицела
- `type` - тип прицела: `cross` (крест), `dot` (точка), `circle` (окружность), `custom` (своё изображение)
- `color` - цвет прицела (lime, red, white, cyan и т.д.)
- `size` - размер прицела в пикселях
- `thickness` - толщина линий
- `gap` - отступ от центра (для креста)
- `custom_image` - путь к своему изображению (используется при type: "custom")

### Примеры настроек

**Крест (по умолчанию):**
```json
{
  "crosshair": {
    "type": "cross",
    "color": "lime",
    "size": 15,
    "thickness": 2,
    "gap": 5
  }
}
```

**Точка:**
```json
{
  "crosshair": {
    "type": "dot",
    "color": "red",
    "size": 8
  }
}
```

**Окружность:**
```json
{
  "crosshair": {
    "type": "circle",
    "color": "cyan",
    "size": 20,
    "thickness": 2
  }
}
```

**Своё изображение:**
```json
{
  "crosshair": {
    "type": "custom",
    "custom_image": "src/custom_crosshair.png"
  }
}
```

Для использования своего изображения положите файл картинки в папку `src/`.

## Экранная лупа

Лупа захватывает область экрана вокруг центра монитора и отображает увеличенное изображение.

### Горячие клавиши
- `magnifier` - показать/скрыть лупу (по умолчанию F1)

### Параметры лупы

```json
{
  "magnifier": {
    "capture_size": 200,
    "display_size": 300,
    "target_fps": 60,
    "interpolation": "linear",
    "offset_x": 200,
    "offset_y": 0,
    "border_color": "#00ff00",
    "border_width": 2
  }
}
```

- `capture_size` - размер захватываемой области (меньше = больше увеличение)
- `display_size` - размер окна лупы
- `target_fps` - целевая частота обновления (адаптируется автоматически)
- `interpolation` - алгоритм масштабирования:
  - `nearest` - пиксельный, самый быстрый
  - `linear` - баланс скорости и качества (рекомендуется)
  - `cubic` - лучше качество, больше нагрузка
  - `lanczos` - максимальное качество, максимальная нагрузка
- `offset_x`, `offset_y` - смещение окна лупы от центра экрана
- `border_color` - цвет рамки
- `border_width` - толщина рамки

## Безопасность

Программа работает как системный overlay:
- Не взаимодействует с процессами игр
- Не читает память других приложений
- Не внедряет DLL
- Использует только стандартные Windows API для захвата экрана (mss)
